- 나눈 횟수
  - 이진수 성질
    - 이진수의 가장 첫 번째 값은 항상 1
    - 숫자 2로 나눈다 라는 것은 오른쪽 자리를 소거하는 것과 동일
    - ex) 1010(10) -> 2로 나누면 101
    - 즉, 나눈 횟수 = 이진수 길이 - 1

- 0부터 시작해서 n씩 점프하면서 length 미만인 것들의 갯수 공식(0, n, 2n, 3n, 4n, ...)
  - ex) length = 6, n = 2
    - 0 ~ 5번 인덱스를 2개로 나눠보면 (0, 1), (2, 3), (4, 5) => 3묶음
    - (전체 길이(length) + n - 1) / 2
  - ex2) length = 7, n = 2
    - (0, 1), (2, 3), (4, 5), (6) => 4묶음 => 3.5묶음
    - length / n이 딱 떨어지면 상관없지만 소수점으로 떨어진다면 올림 연산을 위해 -1을 해줘야 함 
    - Java에서 int 자료형은 소수점을 버리기 때문
    - 식을 다시 정리하자면 size = length / n + (length % n == 0 ? 0 : 1)
