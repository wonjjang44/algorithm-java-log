- 조합
  - 3가지 조합을 찾을 때, i < j < k
  - ex) 1, 2, 3, 4[0, 1, 2, 3]
    - [i, j, k]
      - i는 0과 1
        - i가 0일 때, j는 1 또는 2
          - j가 1일 때, k는 2 또는 3
            - k가 2일 때
              - => [0, 1, 2]
            - k가 3일 때
              - => [0, 1, 3]

          - j가 2일 때, k는 3
            - k가 3일 때
              - => [0, 2, 3]

        - i가 2일 때
          - j는 2
            - j가 2일 때, k는 3
              - => [1, 2, 3]
    - i => 0, length - 2
    - j = i + 1, j < length - 1
    - k = j + 1, k < length


- 소수 찾기
  - 2 ~ n - 1 까지 나눈다
  - ex) 17
    - 17 % 2
    - 17 % 3
    - 17 % 4
    - ...
    - 17 % 16(n -1 => 17 - 1)
    - 이들 중 하나라도 나눠지면(나머지가 0이면) prime number가 아님

  - n/2 까지만 검사
    - n을 나눌 수 있는 약수는 n/2보다 클 수 없다는 성질 이용
    - 2 ~ n / 2 까지 나눈다

  - 약수는 항상 쌍으로 존재한다 => √n(ex_√36(==> 6)) 기준으로 좌우 쌍이 생김
