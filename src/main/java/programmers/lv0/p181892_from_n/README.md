# p181892 - n 번째 원소부터

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181892
- --

## 요약
정수 배열과 정수가 주어질 때, n 번째 원소부터 마지막 원소까지 담은 배열을 리턴하는 메서드 구현.
- --

## 접근
- 배열 복사를 통하여 n부터 기존 배열의 마지막까지의 원소를 뽑아낸다.
- --

## 회고
- 주어진 정수 n 값은 인덱스가 아닌 단순 위치값이라는 것을 인지하자.
  - 해당 값을 인덱스로 사용하려면 -1 처리 필요
- for문을 사용해도 되지만 arraycopy 메서드에 익숙해지기 위해 해당 메서드 사용.
- Arrays.copyOfRange 메서드 사용도 가능할 듯
