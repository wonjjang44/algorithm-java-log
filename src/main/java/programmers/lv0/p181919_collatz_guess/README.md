# p181919 - 콜라츠 수열 만들기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181919
- --

## 요약
모든 자연수 x에 대해서 현재 값이 x이면 x가 짝수일 때는 2로 나누고,<br/>
x가 홀수일 때는 3 * x + 1로 바꾸는 계산을 계속해서 반복하면 언젠가는 반드시 x가 1이<br/>
되는데 해당 과정을 거쳐간 모든 수를 콜라츠 수열이라 함.<br/>
임의의 양의 정수 n을 받아 콜라츠 수열을 리턴하는 메서드 구현 
- --

## 접근
- 조건은 문제에 나와있음.
- 루프를 얼마만큼 돌려야 하는지가 관건임.
- 2가지 방법이 생각남.
  - List 사용
    - ArrayList 사용 시 길이가 동적으로 늘어남
  - 오직 배열만 사용
    - 직접 배열의 크기를 늘리는 로직을 작성해야 함
- n이 1보다 클 때 까지 계속 루프를 돌린다
- -- 

## 회고
- List를 사용하여 수열을 만듦.
- 첫 번째 항은 리스트 생성 직후 바로 넣어준다.
- 순수 배열을 사용해도 되지만, 동적 길이 증가 로직을 추가해야 함
