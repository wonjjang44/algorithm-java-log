# p181905 - 문자열 뒤집기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181905
- --

## 요약
문자열과 두 정수가 주어질 때, s ~ e까지 뒤집은 문자열을 리턴하는 메서드 구현<br/>
ex) Progra21Sremm3, 6, 12<br/>
=> 인덱스 6부터 인덱스 12까지 뒤집은 문자열 > ProgrammerS123 리턴
- --

## 접근
- 방법 1) 루프 돌면서 스왑
- 방법 2) substring 사용
- --

## 회고
- 루프를 돌면서 swap 할 수 있지만 substring을 사용하는게 더 간략한 듯
