# p181927 - 마지막 두 원소

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181927
- --

## 요약
정수 리스트 마지막 원소가 그 전 원소보다 크다면 마지막 원소에서 그 전 원소를 뺀 값을,<br/>
마지막 원소가 그전 원소보다 크지 않다면 마지막 원소를 두 배한 값을 추가하여 return 하는 메서드 구현
- --

## 접근
- 마지막 원소(배열 전체 길이 -1)가 그 전 원소(배열 전체 길이 - 2)값과 비교하는 연산
- 파라미터 배열 길이만큼 반복하면서<br/>
  마지막 원소와 그 전 원소의 대소를 비교하여 뺄셈 연산 혹은 곱셈 연산을 할지 정한다
- -- 

## 회고
- 위 접근법처럼 적용했을 때, num_list 배열의 마지막 값이 덮어 씌워짐.
- 기존 num_list의 길이 -1 => num_list 배열의 마지막 값
- 그렇기에 배열이의 길이가 num_list 길이보다 1 큰 새로운 배열이 필요하다고 판단
- 수의 대소를 구분하기 전 num_list의 길이 + 1한 배열을 생성한다.
- 이후 새로 생성한 배열에 num_list 배열의 값을 할당한다.
  - System.arraycopy() 메서드를 사용해도 될 듯
