# p181913 - 문자열 여러 번 뒤집기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181913
- --

## 요약
이차원 배열로 주어진 값은 [s, e] 형태로 my_string의 인덱스 s부터 e까지 뒤집은 문자열을 리턴하는 메서드 구현
- --

## 접근
- 문자열을 배열로 변환 후 반복하면서 (0 - 7), (1 - 6), (2 - 5), (3 - 4) 꼴로 반복.
- 반복 횟수는 start가 end보다 작을 때 까지 반복(start는 1씩 더해지고 end는 1씩 감소함)
- --

## 회고
- 이차원 배열을 [s, e] 꼴로 나눈 후 반복하면서 문자열을 스왑하려고 함.
- 스왑하면서 첫 번째와 마지막 스왑 이후(예를들어 [0, 7]이면 0과 7 스왑, 1과 6스왑 꼴) 위치를 변경해주지 않았음.
- 또한 문자열을 배열로 변환하는 로직을 for문 내부에 작성하여 위치를 변경해도 my_string 문자열을 다시 배열로 변환하기 때문에 스왑 적용이 안 됐음.
  - 문자열을 배열로 만드는 로직을 for문 밖으로 뺌
  - while문으로 변경하고 start를 1씩 더해주고, end를 1씩 감소시킴(start++, end--)
