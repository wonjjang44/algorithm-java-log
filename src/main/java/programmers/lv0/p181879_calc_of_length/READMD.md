# p181879 - 길이에 따른 연산 

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181879
- --

## 요약
정수 배열이 주어질 때, 배열의 길이가 11 이상이면 모든 원소들의 합을,<br/>
10 이하면 모둔 원소의 곱을 리턴하는 메서드 구현
- --

## 접근
- 길이에 따른 조건 분기하면 될 듯
- --

## 회고
- 배열 길이가 10 이하면 모든 원소의 곱을 담아야 함.
  - 기존 answer 변수의 값은 0이 할당돼 있음. => 0을 곱하면 언제나 0이 나옴
  - 따라서 삼항 연산자를 통하여 기존값을 산정한다.
    - 길이가 11이상이면 0, 이하면 1
