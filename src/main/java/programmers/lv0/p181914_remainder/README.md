# p181914 - 9로 나눈 나머지

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181914
- --

## 요약
음이 아닌 정수를 9로 나눈 나머지는 그 정수의 각 자리 숫자의 합을 9로 나눈 나머지와 같음을 이용하여<br/>
음이 아닌 정수가 문자열 number로 주어질 때, 이 정수를 9로 나눈 나머지를 return 하는 메서드 구현
- --

## 접근
- 문자열로 이루어진 숫자를 "" 기준으로 하나씩 잘라 String[] 에 담는다.
- 루프를 돌면서 int 타입 변수에 누적한다.
- 누적된 값을 9로 나눈 나머지를 리턴해준다.
- --

## 회고
- 최초 문자열을 잘라 배열로 만든다는 생각은 했었으나 split(""), 즉 "" 기준으로 나눌 생각을 못했음.
