# p002 - 각도 합치기(PCCE 기출문제 2번)

## 요약
좌표 평면에서 두 선분이 이루는 각도는 한 바퀴를 360도로 표현한다.
360도와 30도는 같은 각도인 셈.
두 각을 나타내는 angle1, angle2 변수가 주어질 때, 두 각의 합을 0 <= angle1 + angle2 < 360으로
출력하는 코드를 작성할 것.

입력 예시)
280
485

출력 예시)
45 
=> 280 + 485 = 765 - 720 = 45
- --

## 접근
- 각도의 합이 360
- --

## 체크 포인트
- 각도의 합인 360의 배수의 나머지
- --
