## 요약
정수 배열이 주어질 때, 각 원소에 대하여 값이 50보다 크거나 같고, 동시에 짝수라면 2로 나누고,<br/>
50보다 작고, 홀수라면 2를 곱한 배열을 리턴하는 메서드 구현.
- --

## 접근
- 조건에 맞게 &&로 분기하여 처리
- --

## 회고
- 첫 번째 조건을 50보다 크고, 동시에 짝수일 때 라고 해석함.
