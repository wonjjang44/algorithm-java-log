## 요약
정수 l 이상 r이하의 정수 중 "0"과 "5"로만 이루어진 오름차순 배열을 리턴하는 메서드 구현
만약, 정수가 없을 시 -1이 담긴 배열을 리턴한다
- --

## 접근
- 파라미터로 받은 값의 사이 값을 빈 배열에 추가한다.
- int 배열을 String으로 변경하고 contains를 사용하여 "0", "5"가 포함된 값을 출력
- -- 

## 회고
- contains로 뽑으려고 하니 제대로 필터링 되지 않음.
- 문자(char) '0'과 '5'를 조건으로 부여하자
- 다시 한 번 더 풀어봐야 함
