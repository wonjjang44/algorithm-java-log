## 요약
정수 배열과 정수가 주어질 때, 정수 배열을 n 번째 원소 이후의 원소들과 n 번째까지의 원소들로 나눠<br/>
n 번째 원소 이후의 원소들을 n 번째까지의 원소들 앞에 붙은 배열을 리턴하는 메서드 구현
- --

## 접근
- 두 개의 배열을 만들어서 조건대로 자른 뒤 병합
- --
