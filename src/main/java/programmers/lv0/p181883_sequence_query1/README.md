# p181883 - 수열과 구간 쿼리1

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181883
- --

## 요약
정수 배열과 정수 2차원 배열이 주어질 때, 2차원 배열의 원소는 [s, e] 꼴.<br/>
s <= i <= e인 모든 i에 대하여 arr[i]에 1을 더한 배열을 리턴하는 메서드 구현.
- --

## 접근
- 2차원 배열의 원소의 [s, e] 꼴을 반복하여 뽑아낸 뒤, 다시 s부터 e까지 루프를 돌면서 조건(s <= i <= e)에
  맞는 i를 찾고 arr[i]에 + 1 한다.
- --

## 회고
- 내부 for문을 돌면서 시작값과 범위를 미리 구한 s와 e로 대체.
