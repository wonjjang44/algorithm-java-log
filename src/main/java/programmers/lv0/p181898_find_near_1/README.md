# p181898 - 가까운 1 찾기 

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181898
- --

## 요약
정수 배열이 주어질 때, 해당 배열의 원소는 1 또는 0.<br/>
정수값이 주어질 때, 해당 정수 값보다 크면서 배열의 값이 1인 가장 작은 인덱스를 리턴하는 메서드 구현.<br/>
없다면 -1 리턴.
- --

## 접근
- 루프를 주어진 정수부터 시작한다면 파라미터로 받은 정수값 보다 크면서 배열의 값이 1인 가장 작은 인덱스 조건에 성립한다.
- --

## 회고
- 조건에 맞는 값을 찾았을 경우 break를 걸어 빠져나와야 함.
