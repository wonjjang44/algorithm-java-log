# p181878 - 원하는 문자열 찾기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181878
- --

## 요약
알파벳 문자열이 주어질 때, 연속된 부분 문자열 중 파라미터로 받은 특정 문자열이 존재하면 1,<br/>
존재하지 않는다면 0을 리턴하는 메서드 구현
- --

## 접근
- 슬라이딩 윈도우 알고리즘 적용
- 파라미터로 받은 문자열(pat)을 한 칸씩 밀면서 기존 myString 값과 비교
- --

## 회고
- 슬라이딩 윈도우 방식은 이중 루프를 돌기 때문에 O(N^2)
- indexOf 또는 contains 사용 시 간략하게 할 수 있을 듯
