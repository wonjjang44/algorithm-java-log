# p181886 - 5명씩

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181886
- --

## 요약
문자열 배열이 주어질 때, 앞에서 부터 5명씩 묶은 그룹의 가장 앞에 서있는 사람들의<br/>
이름을 담은 배열을 리턴하는 메서드 구현<br/>
단, 마지막 그룹이 5명이 되지 않아도 가장 앞에 있는 사람의 이름을 포함할 것.
- --

## 접근
- n은 5로 고정
- 그룹 개수는 length / n + (length % n == 0 ? 0 : 1)
- --

## 회고
- 나머지 연산 사용
- 나머지 연산 후 배열의 크기를 Arrays.copyOf() 메서드로 잘라도 됨
- 또는 그룹 개수 공식 사용 가능(size = length / n + (length % n == 0 ? 0 : 1))
