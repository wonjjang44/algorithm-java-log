# p181900 - 글자 지우기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181900
- --

## 요약
문자열과 정수 배열이 주어질 때, 문자열에서 정수 배열의 원소에 해당하는<br/>
인덱스의 값을 지우고 이어 붙은 문자열을 리턴하는 메서드 구현.
- --

## 접근
- List에 담고 루프를 돌면서 list.remove() 메서드를 사용한다.
- --

## 회고
- List에 담고 루프를 돌면서 list.remove() 메서드를 사용했지만 리스트의 값이 지워지면서
  인덱스가 초기화 되는 문제가 생김.
- 정수 배열의 인덱스에 해당하는 값은 공란(' ')으로 할당한 뒤, 배열을 replaceAll하여 공란을 제거함.
