# p181931 - 등차수열의 특정한 항만 더하기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181931
- --

## 요약
첫째항이 a, 공차가 d인 등차수열에서 included[i]가 i + 1항을 의미할 때,<br/>
이 등차수열의 1항부터 n항까지 included가 true인 항들만 더한 값을 리턴
- --

## 접근
- 첫 번째 항과 공차가 제공됐으므로 등차수열 배열을 만들 수 있다.
- included boolean 배열의 값이 true일 때, true 값을 가지는 인덱스를 추출하여 새롭게 정의한<br/>
  등차 수열의 배열의 인덱스에 해당하는 값을 answer 변수에 누적한다.
- -- 

## 회고
- 첫 번째 항을 등차 수열의 첫 번째 항에 할당하는 로직이 누락되어 seq 정수 배열의 0 번째
    인덱스에 첫 번째 항 + 공차 값이 들어갔었음.
- 등차 수열 배열을 만들었지만 included boolean 배열의 값이 true일 때, true 값을 가지는 인덱스를<br/> 
  추출하는 부분에서 시간을 많이 소요했다.
- 배열 루프를 돌릴 때, 하나의 배열만 계속 바라보는 것이 아닌 여러 갈래로 생각을 해봐야 할 것 같다.
