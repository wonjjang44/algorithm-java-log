# p181884 - n보다 커질 때까지 더하기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181884
- --

## 요약
정수 배열과 정수가 주어질 때, 정수 배열의 모든 원소를 더하다가<br/>
그 합이 n보다 커지는 순간 지금까지 더한 원소들의 합을 리턴하는 메서드 구현
- --

## 접근
- 루프를 돌면서 조건에 맞을 때, break하여 누적 합계를 리턴
- --

## 회고
- for문 내부에서 if문으로 조건을 주기보단 for문 자체에 조건을 부여하여 해결할 수 있을 듯
  - ex) for(int i = 0; i <= n; i++)
