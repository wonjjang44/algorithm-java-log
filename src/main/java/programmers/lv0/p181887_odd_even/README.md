# p181887 - 홀수 vs 짝수

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181887
- --

## 요약
정수 배열이 주어질 때, 가장 첫 번째 원소를 1번 원소라고 할 때, 홀수 번째 원소들의<br/>
합과 짝수 원소들의 합 중 큰 값을 리턴하는 메서드 구현(두 값이 같을 경우 그 값 리턴)
- --

## 접근
- 홀/짝 조건 분기
- --

## 회고
- 배열에서는 0번 인덱스가 가장 첫 번째 원소지만 문제에서 0번 인덱스를 1번으로 치환함
- 여기서 혼동이 있었던 것 같음
- 결국 0번째 인덱스가 1번 원소 => 반복문에서 numList[0] => 홀수, numList[1] => 짝수
