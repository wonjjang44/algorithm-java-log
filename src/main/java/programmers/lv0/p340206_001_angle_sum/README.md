# p002 - 각도 합치기(PCCE 기출문제 2번)

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/340206
- --

## 요약
좌표 평면에서 두 선분이 이루는 각도는 한 바퀴를 360도로 표현한다.
360도와 30도는 같은 각도인 셈.
두 각을 나타내는 angle1, angle2 변수가 주어질 때, 두 각의 합을 0 <= angle1 + angle2 < 360으로
출력하는 코드를 작성할 것.
- --

## 접근
- 각도의 합이 360도 이므로 360의 배수의 나머지 값
- --

## 회고
- 각의 합이 360이라고 주어졌을 때, 360의 배수의 나머지가 떠올랐다.
