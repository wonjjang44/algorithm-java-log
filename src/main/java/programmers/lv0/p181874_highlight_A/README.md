# p181874 - A 강조하기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181874
- --

## 요약
문자열이 주어질 때, 소문자 "a"가 등장하면 대문자 "A"로 변환하고<br/>
"A"가 아닌 모든 대문자 알파벳은 소문자로 변경하여 리턴하는 메서드 구현
- --

## 접근
- 루프를 돌면서 특정 문자열을 변경하는 것이므로 replace 메서드 사용해도 될 듯?
- 아니면 루프 돌면서 조건에 맞춰 변경해주는 방식도 고려
- --

## 회고
- replaceAll 메서드 사용 시 정규식은 잡는게 애매함
- 문자열을 쪼개어 로프를 돌면서 소문자 a일 땐 Upper, 아니라면 Lower 후 할당했더니 테스트 케이스에서 걸림
  - 테스트 문자열 abstract algebra 에서 만약 가장 첫 번째 문자열이 a가 아닌 A라면?
  - abstrAct AlgebrA 리턴함
  - 최초 문자열을 모두 소문자로 변경한 뒤 시도하면 대문자 A로 변경됐다가 다시 소문자 a로 돌아갈 경우가 사라짐
