# p181888 - n개 간격의 원소들

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181888
- --

## 요약
정수 배열과 정수가 주어질 때, 정수 배열의 첫 번째 원소부터 마지막 원소까지<br/>
n개 간격으로 저장돼 있는 원소들을 차례로 담은 배열을 리턴하는 메서드 구현
- --

## 접근
- 새로운 배열의 크기는 파라미터로 받은 배열의 크기로 선정한다.
- 계산 이후 리턴 시 배열의 빈 공간은 Arrays.copyOf() 메서드로 솎아낸다.
- --

## 회고
- Arrays.copyOf() 메서드로 자르려고 했으나 중간 중간 0이 포함돼 있어서 자르기 애매했음.
  - => [4, 0, 6, 0, 7, 0]
- 0을 어떻게 솎아낼지 고민을 많이 한 것 같음
  - 중간에 위치한 0을 골라내기엔 로직이 더 추가되므로 값들을 적재한다.
    - 즉, [4, 0, 6, 0, 7, 0] ==> [4, 6, 7, 0, 0, 0]
    - 이 상태에서 Arrays.copyOf()를 사용하면 7 이후 값을 깔끔하게 잘라낼 수 있음
  - 최종 길이를 알 수 있다면 로직이 좀 더 간결해질 수 있을텐데..
    - size = length / n + (length % n == 0 ? 0 : 1)
