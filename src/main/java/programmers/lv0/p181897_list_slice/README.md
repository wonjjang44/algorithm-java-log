# p181897 - 리스트 자르기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181897
- --

## 요약
정수와 정수 3개가 담긴 배열, 정수 N개가 담긴 배열이 주어질 때, 정수값에 따라 배열을 자르려고 한다.<br/>
> - n = 1 : num_list의 0번 인덱스부터 b번 인덱스까지
> - n = 2 : num_list의 a번 인덱스부터 마지막 인덱스까지
> - n = 3 : num_list의 a번 인덱스부터 b번 인덱스까지
> - n = 4 : num_list의 a번 인덱스부터 b번 인덱스까지 c 간격으로

위 조건을 만족하는 배열을 리턴하는 메서드 구현.
- --

## 접근
- 조건이 명확하게 주어졌기 때문에 switch문을 사용하여 분기한다.
- 주어진 answer 배열을 각 n 값에 따라 각기 다른 길이를 부여한다.
- 문제에 주어진 조건을 그대로 표현한다.
- --

## 회고
- for문으로 배열을 복사하는 대신 System.arraycopy 메서드를 사용하여 배열을 복사했다.
- n이 4일 때의 조건식은 a번째 인덱스 ~ b번째 인덱스까지 c 간격 인 조건.
  - 이 조건에서 막혔음. 
  - 총 구간의 길이는 b번째 인덱스 - a번째 인덱스
  - c 간격은 (b번째 인덱스 - a번째 인덱스) / c 간격
  - a 인덱스와 b 인덱스 값이 포함이므로 + 1
  - 즉, (b번째 인덱스 - a번째 인덱스) / c 간격 + 1
