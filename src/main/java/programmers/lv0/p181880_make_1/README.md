## 요약
정수가 있을 때, 짝수라면 반으로 나누고,<br/>
홀수라면 1을 뺀 뒤 반으로 나누면 마지막엔 1이 된다.<br/>
ex)
> - 10 / 2 = 5
> - (5 - 1) / 2 = 2
> - 2 / 2 = 1

위와 같이 3번의 연산으로 값이 1이 됨.<br/>
정수 배열이 주어질 때, 정수 배열의 모든 원소를 1로 만들기 위해 필요한 연산 횟수를 리턴하는 메서드 구현
- --

## 접근
- 배열 원소 만큼 반복하면서 원소마다 홀/짝 비교 후 연산하여 값이 1보다 클 때 까지 계산
- --

## 회고
- 다른 방식으로 이진수의 성질을 이용한 방법이 있음
  - 정수가 1이 될 때까지 2로 나눈 횟수의 총합을 구하는 것이 핵심
  - 어떤 수를 2로 나눈다 => 이진수 값의 맨 앞 1만 남길 때까지 오른쪽 값을 소거하는 것
  - ex)
    - 1000(8) => 길이 = 4, 나눈 횟수 = 3
    - 1010(10) => 길이 = 4, 나눈 횟수 = 3
    - 101(5) = 길이 = 3, 나눈 횟수 = 2
  - 즉 1000(8)에서 맨 앞 1만 남도록 1 기준 오른쪽 값을 소거한 횟수가 2로 나눈 횟수
