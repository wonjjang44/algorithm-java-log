# p181909 - 접미사 배열

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/181909
- --

## 요약
어떤 문자열에 대해서 접미사는 특정 인덱스부터 시작하는 문자열을 의미함.<br/>
ex) "banana"의 모든 접미사는 "banana", "anana", "nana", "ana", "na", "a"<br/>
파라미터 my_string의 모든 접미사를 사전순으로 정렬한 문자열 배열을 return하는 메서드 구현
- --

## 접근
- "" 기준으로 자르면 안 되고 파라미터로 받은 문자열의 전체 길이에서 하나씩 감소하면서 넣어야 함
- 마지막엔 정렬

## 회고
- 단순 "" 기준으로 자르고 배열에 넣는 것을 생각했지만 잘못된 생각.
- 파라미터로 받은 문자열의 길이 만큼 루프를 돌면서 문자열을 substring한다.
- 0번째 부터 문자열의 총 길이까지 시작 위치만 +1씩 증가하여 자른 뒤, 배열에 넣는다.
- 배열이므로 Arrays 유틸 클래스의 sort() 메서드를 호출하여 정렬한다
