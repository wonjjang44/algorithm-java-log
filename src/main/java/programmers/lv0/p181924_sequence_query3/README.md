## 요약
정수 배열 arr와 2차원 정수 배열 queries이 주어질 때,<br/>
queries의 원소는 각각 하나의 query를 나타내며, [i, j] 꼴임<br/>
query마다 순서대로 arr[i]의 값과 arr[j]의 값을 서로 바꾼다.
arr 배열을 리턴하는 메서드를 구현할 것
- --

## 접근
- 2차원 배열 queries [[0, 3], [1, 2], [1, 4]]이 존재한다고 가정
- queries[0] = [0, 3]
- queries[1] = [1, 2]
- queries[2] = [1, 4]
- 기존 1차원 배열의 0번째 원소와 3번째 원소의 자리를 변경한다.
  - temp 변수 선언 후 기존 값을 넣어 두고 변경한 다음 다시 해당 값으로 엎어친다.
  - 이런식으로 1차원 배열의 자리를 변경한 후 리턴한다.
- -- 

## 회고
- 자리 바꾸기 문제인 듯
- 접근은 했는데 이중 for문을 사용하다가 막힘.
- 너무 꼬아서 생각한 듯 함.
- 단순 앞뒤 자리를 변경하면 되므로 0번째, 1번째 값을 저장한 뒤,<br/>
  이들 값을 사용하여 arr 값을 자리 변경해주면 됨.
- 도움 받은 문제이므로 다시 풀어볼 것
