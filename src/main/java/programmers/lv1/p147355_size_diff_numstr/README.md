# p147355 - 크기가 작은 부분문자열

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/147355
- --

## 요약
숫자로 이루어진 문자열 t, p가 주어질 때, t에서 p와 길이가 같은 부분문자열 중,<br/>
부분문자열이 나타내는 수가 p가 나타내는 수 보다 작거나 같을 횟수를 리턴하는 메서드 구현.

<제한사항><br/>
1 ≤ p의 길이 ≤ 18<br/>
p의 길이 ≤ t의 길이 ≤ 10,000<br/>
t와 p는 숫자로만 이루어진 문자열이며, 0으로 시작하지 않는다.
- --

## 접근
- 몇 번 반복할지가 중요.
- 반복 횟수 : p.length - t.length + 1
- t를 i부터 시작해 p의 길이에 + i 한 만큼 잘라낸다
- 잘라낸 값과 p를 비교하여 p가 크거나 같다면 + 1
- -- 

## 회고
- 반복 횟수는 p길이에서 t 길이를 뺀 다음 + 1
- t를 잘라내는데 i(0~) 시작해서 p길이 + i(0 ~)까지 잘라낸다.
