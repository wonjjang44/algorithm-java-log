# p77884 - 약수의 개수와 덧셈

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/77884

## 요약
left ~ right 까지의 모든 수들 중, 약수의 개수가 짝수인 수는 더하고, 홀수인 수는 뺀 수를 리턴하는 메서드 구현.
- --

## 접근
- 루프 범위를 left ~ right까지 설정한다.
- 약수는 자기 자신을 1부터 자기 자신까지의 수로 나눈 나머지가 0인 값들.
  - 이중 루프 필요
- -- 

## 회고
- 약수를 찾아 cnt하고, cnt가 짝수(% 2 == 0)라면 더하고, 아니라면 뺀다.
- 다른 방법으로는 이중 루프를 돌 필요 없이 제곱근을 이용하여 풀 수 있다.
  - 제곱수인 경우 약수의 개수가 홀수개가 된다.(자기 자신과 짝이 되는 경우가 있으므로 반드시 홀수개가 됨)
  - ex) 16의 약수
    - 16의 제곱근은 4
    - 16의 약수는 1, 2, 4, 8, 16
    - (1 * 16), (2 * 8), (4 * 4) => 4 * 4 즉, 자기 자신과 짝이 되므로 약수의 개수는 홀수개가 됨
  - 이러한 성질을 이용하면 로직이 간단해짐
  - i % Math.sqrt(i) == 0 => 홀수개이므로 answer -= i; 제곱근이 아니라면 약수의 개수는 짝수개이므로 answer += i;
