# p86051 - 없는 숫자 더하기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/86051
- --

## 요약
0부터 9까지의 숫자 중 일부가 들어있는 정수 배열 numbers가 매개변수로 주어질 때,<br/>
numbers에서 찾을 수 없는 0 ~ 9까지의 숫자를 모두 찾아 더한 값을 리턴
- --

## 접근
- 루프를 돌면서 0 ~ 9 의 숫자 배열과 비교하여 없는 숫자를 판별한다.
  - numbers 배열의 원소들과 0 ~ 9 의 숫자 배열을 비교해야 하므로 이중 루프를 돌아야 함.
- 없는 수를 answer변수에 누적한다.
- -- 

## 회고
- 이중 루프를 돌기 전 numbers 배열을 오름차순으로 정렬을 해야 함
  - 0 ~ 9까지의 수도 오름차순으로 정렬돼 있기 때문에 정렬을 맞춰주고 비교하는 것이 나을 것이라고 판단
- 비교를 할 때, 자릿수도 맞춰줘야 함.
  - 0 ~ 9까지의 수가 10개 이므로 numbers도 10개로 원소를 맞춰줘야 함. 없는 값은 -1로 대체.
- 루프를 돌리다보니 데이터가 일치하는 않는 위치 기준으로 전부 false가 떨어짐.
  - 원소들을 오른쪽으로 밀어줘야 함(시프트 연산 필요)
