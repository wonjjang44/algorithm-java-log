# p68644 - 두 개 뽑아서 더하기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/68644
- --

## 요약
정수 배열 numbers에서 서로 다른 인덱스에 있는 두 개의 수를 뽑아서<br/>
더하여 만들 수 있는 모든 수를 배열에 오름차순으로 담아 리턴하는 메서드 구현.
- --

## 접근
- 배열의 0번 째 인덱스에 해당하는 값과 1번 째에 해당하는 값을 더한다.
- 계속해서 0번 째와 2번 째, 0번 째와 3번 째, .. 이런식으로 더해간다.
- 즉, numbers[i] + numbers[i + 1]
- -- 

## 회고
- 이중 루프를 돌려서 numbers 배열의 0번 째 인덱스 값과 1번 째 인덱스 값을 더해간다.
- 이중 루프를 돌다보면 중복된 값이 생기는데 이를 해결하기 위해 Set에 담는다.
- 마지막으로 Set에 담은 값들을 배열에 담은 뒤, 오름차순 정렬한다.
