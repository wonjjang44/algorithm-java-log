# p131705 - 삼총사

## 요약
학생 3명의 정수 번호를 더했을 때, 0이 되면 3명의 학생은 삼총사라고 한다.<br/>
ex)
> 5명의 학생이 존재하고 각 정수 번호는 -2, 3, 0, 2, -5일 때<br/>
> 첫 번째, 세 번째, 네 번째 학생의 정수 번호를 더하면 0이므로 세 학생은 삼총사.<br/>
> 또한 두 번째, 네 번째, 다섯 번째 학생의 번호를 더해도 0이므로 삼총사.

정수 배열 number가 주어질 때, 삼총사를 만들 수 있는 방법의 수를 리턴하는 메서드 구현
