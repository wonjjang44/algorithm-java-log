# p132267 - 콜라 문제

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/132267
- --

## 요약
빈 병 20병을 가져가서 콜라 10병을 받는다. 받은 10병을 모두 비운 뒤, 다시 가져가 콜라 5병을 받는다.<br/>
콜라 5병 중 4병을 비우고, 빈 병 4병을 가져가 다시 콜라 2병을 받는다. 다시 콜라 2병을 모두 비운뒤<br/>
가져가 콜라 1병을 받는다. 위 남은 콜라 1병과 새롭게 받은 콜라 1병을 다시 비운뒤, 가져가 콜라 1병을 다시 받는다.<br/>
결국 콜라 10 + 5 + 2 + 1 + 1 총 19병을 받을 수 있다.<br/>
이를 일반화하여 빈 병 a개를 가져다주면 b병을 리턴할 때, 빈병 n개를 가져다주면 콜라 몇 병을 받을 수 있는지<br/>,
즉 콜라 개수를 리턴해주는 메서드 구현.<br/>
단, 보유 중인 빈 병이 a개 미만이면, 추가적으로 빈 병을 받을 수 없다.<br/>
a = 콜라를 받기 위해 마트에 가져다줘야 하는 빈 병 수<br/>
b = 빈 병 a개를 받고 콜라를 리턴해주는 수<br/>
n = 보유 중인 빈 병의 개수
- --

## 접근
- 리턴받을 콜라 개수는 빈 병의 개수 / 마트에 가져다준 빈 병 개수 * 리턴해줄 콜라 개수
- 마시지 않은 콜라, 즉 나머지 값도 들고 있어야 힘
- -- 

## 회고
- 빈 병의 개수(n) / 마트에 가져다준 빈 병 개수(a) 로 계산하고 
  해당 계산에서 발생된 나머지도 같이 구하여 서로 더해 누적하는 식으로 계산했어.
- 잘못된 값이 출력됐어.
  - 새로 받는 콜라 개수는 (n / a) * b
  - b가 1보다 큰 경우를 생각하지 않은거지
