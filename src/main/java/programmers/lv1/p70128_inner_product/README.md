01.22 문제
# p70128 - 내적 

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/70128
- --

## 요약
길이가 서로 같은 1차원 정수 배열 a, b에 대하여 내적을 return 하는 메서드 구현
- --

## 접근
- 내적은 행렬의 곱 같은 형식.
  - a[0]*b[0] + a[1]*b[1] + ... + a[n-1]*b[n-1]
- -- 

## 회고
- a 또는 b 배열만큼 루프를 돌면서 answer에 a, b 같은 자리(같은 인덱스)에 해당 하는 값을 서로 곱한 뒤, 누적한다.
