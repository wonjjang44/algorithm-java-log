# p87389 - 나머지가 1이 되는 수 찾기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/87389
- --

## 요약
자연수 n을 x로 나눈 나머지가 1이 되도록 하는 가장 작은 자연수 x를 리턴.
- --

## 접근
- n % x == 1 값을 만족하는 수에서 가장 작은 값을 출력하는 문제
- 루프는 2부터 시작. 1로 나눈 나머지는 항상 0이 되므로 1은 제외
- 
- -- 

## 회고
- 반복 범위를 잘못 잡음
- n % x 에서 나머지 값은 항상 0 ~ (x - 1) 사이로 떨어짐
- n % x에서 x는 n보다 클 필요가 없음.
  - x가 n보다 크다면?
    - 10 % 12 = 10
    - 10 % 100 = 10
    - x > n => n % x = n
- 따라서 n % x == 1이 성립하려면 x < n 가 돼야 함
