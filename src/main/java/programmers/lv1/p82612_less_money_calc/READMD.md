# p82612 - 부족한 금액 계산하기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/82612
- --

## 요약
놀이기구의 기존 이용료 = price원. 놀이기구를 N번 째 이용한다면 기존 price원에 N배.<br/>
처음 이용료가 100이라면 두 번째는 200, 세 번째는 300으로 인상.<br/>
놀이기구를 count번 타면 현재 자신이 가지고 있는 금액(money)에서 얼마가 모자라는지 리턴해보자.<br/>
단, 금액이 모자르지 않다면 0 리턴.
- --

## 접근
- ex) 처음 이용료 : 3 / 현재 보유한 금액 : 20 / 이용 회수: 4
- (3 * 1) + (3 * 2) + (3 * 3) + (3 * 4) = 30 - 20 = 10
- price * (1 + 2 +3 + 4)
