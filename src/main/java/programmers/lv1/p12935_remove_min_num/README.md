# p12935 - 제일 작은 수 제거하기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/12935
- --

## 요약
정수 배열에서 가장 작은 수를 제거한 뒤 리턴하는 메서드 구현.<br/>
단, 리턴할 배열이 빈 배열이라면 -1을 넣은 뒤 리턴.
- --

## 접근
- 루프를 돌면서 최솟값을 찾아냄
- 비교할 대상이 없으면 -1을 넣어 리턴해야 함
- -- 

## 회고
- 배열 리스트 자료구조 사용할 때, remove() 메서드를 사용하여 최솟값
- 인자값으로 int형 값을 넘기면 list의 인덱스에 해당하는 값을 제거함
- 하지만 객체를 넘기면 그 값 자체를 제거함
- remove() 메서드에 정수값을 넘겨서 IndexOutOfBoundsException이 터지고 있었음.
  - ex) new int[]{10}을 넘길 경우 해당
- Integer.valueOf()로 인덱스가 아닌 값 자체를 제거하는 식으로 변경
- 너무 많은 for문이 사용된 것 같음..
- 비효율적인 로직이 포함된 듯.
- 추후 stream으로 다시 풀어볼 예정
