# p12940 - 최대공약수와 최소공배수

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/12940
- --

## 요약
입력 받은 두 수의 GCD와 LCM을 리턴하는 메서드 구현
- --

## 접근
- 두 수를 1부터 9까지 나눈다
- 2개의 수가 더 이상 나누어 떨어지지 않을 때 까지 반복한 값 중 제일 큰 값이 GCD이며 LCM은 GCD * 몫
- -- 

## 회고
- 테스트 9, 11, 13, 15가 실패로 떨어짐
- 루프 범위를 9로 고정한 것이 문제가 되는 것 같음.
  - 최대공약수는 작은 수보다 클 수 없음.
    - ex) 12 18 => 18을 나눌 수 있는 수가 18 초과일 수는 없음
  - 12와 18을 공통적으로 나눌 수 있는 수는 12 안에 있기 때문에 반복 제한을 min(n, m)으로 둔다.
