# p12917 - 문자열 내림차순으로 배치하기

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/12917
- --

## 요약
문자열(대소문자) s를 큰것부터 작은 순으로 정렬(내림차순)하여 리턴하는 메서드 구현.<br/>
단, 대문자는 소문자보다 작은 것으로 간주
- --

## 접근
- char 타입 사용(소문자가 대문자보다 크다고 했으므로)
- 루프를 돌면서 스왑 예정 
- -- 

## 회고
- 선택정렬 사용
- 내림차순이므로 기준값보다 비교 대상이 더 클 때 스왑
