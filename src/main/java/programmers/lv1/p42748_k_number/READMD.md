# p42748 - K번째수

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/42748
- --

## 요약
배열 array, [i, j, k]를 원소로 가진 2차원 배열 commands가 매개변수로 주어질 때,<br/>
아래 조건에 맞는 배열을 리턴하는 메서드 구현<br/>

[조건]<br/>
array가 [1, 5, 2, 6, 3, 7, 4], i = 2, j = 5, k = 3 이라면<br/>
array의 2번째부터 5번째까지 자르면 [5, 2, 6, 3]<br/>
1에서 나온 배열을 정렬하면 [2, 3, 5, 6]<br/>
2에서 나온 배열의 3번째 숫자는 5.
- --

## 접근
- substring 또는 split 사용하여 문자열 분리 후 시도하려고 했지만 더 복잡해져서 다른 방법 구상.
- 조건에 맞는 배열을 원본 배열에서 복사하여 구현함
- -- 

## 회고
- 이중 for문을 사용하여 조건에 맞는 배열을 조작하려 했음.
- 최근 새롭게 알게된 System.arraycopy() 메서드를 사용하여 이중 for문을 사용하지 않고 배열을 복사함.
- Arrays.copyOfRange() 메서드 사용도 괜찮을 듯
