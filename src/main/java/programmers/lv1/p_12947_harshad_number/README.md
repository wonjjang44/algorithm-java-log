# p12947 - 하샤드 수

- 링크: https://school.programmers.co.kr/learn/courses/30/lessons/12947
- --

## 요약
양의 정수 x가 하샤드 수이려면 x의 자릿수의 합으로 x가 나누어져야 한다.<br/>
ex)
> - 18의 자릿수 합은 1+8=9이고, 18은 9로 나누어 떨어지므로 18은 하샤드 수
 
자연수 x를 받아 x가 하샤드 수인지 판별하는 메서드 구현
- --

## 접근
- 자릿수를 구하기 위해 정수를 문자열로 변환한 다음 배열로 뽑아낸다.
- 이후 각 자릿수를 누적한다.
- 이 값이 기존 x와 % 연산을 진행하여 값이 0이라면 하샤드 수
- -- 

## 회고
- 자릿수를 구하기 위한 방법이 여러가지일 듯
- 문자열로 변환 후 각 자릿수를 뽑아내는 것이 간단한 것 같아 사용
