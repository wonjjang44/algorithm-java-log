# Java 알고리즘 문제 풀이 및 기록
- **풀이 누적 + 실수 패턴 확인 및 보완**
- ---

## 목표
- 1일 1~2문제
- 틀린 문제는 3일 뒤 재도전한다
- 문제마다 README.md를 작성한다
  - ### 문제별 README 템플릿
    - 각 문제 폴더의 `README.md`에는 다음과 같이 기록한다:
      - 접근 방법(왜 이렇게 풀었는가)
      - 핵심 포인트(조건/경계값/함정)
      - 시간복잡도
      - 회고(내가 한 실수 & 다음엔 어떻게 피할지)
- ---

## mistakes.md
- 문제를 풀다 반복해서 틀리는, 실수하는 패턴을 누적 기록
- ---

## Commit
- 문제 1개 = 커밋 1개

## 디렉토리 구조
- notes / mistakes.md
  - 자주 실수하는 패턴 기록
- 플랫폼-난이도-문제번호(pk)_slug(문제 영어로 짧게 요약해서)
  - ex) programmers.lv0.p340206_001_angle_sum.Solution
